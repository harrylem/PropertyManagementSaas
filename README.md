# Koinoxrista — Property Management Prototype

## Project Structure
```
koinoxrista/
├── package.json          # Tailwind dependency + build scripts
├── tailwind.config.js    # Custom theme (colors, fonts)
├── vercel.json           # Vercel deployment config
├── src/
│   └── input.css         # Tailwind directives + custom styles
└── public/
    ├── index.html         # The app (all 9 pages)
    └── style.css          # ← Generated by build (don't edit)
```

## Deploy to Vercel

### Option A: Push to GitHub → connect to Vercel
1. Push this folder to a GitHub repo
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repo → Vercel auto-detects the config → Deploy

### Option B: CLI
```bash
cd koinoxrista
npm install
npm run build        # generates public/style.css
npx vercel           # deploys to Vercel
```

## Local Development
```bash
npm install
npm run dev          # watches for changes, rebuilds CSS
# Open public/index.html in browser
```

## Pages
- **Πίνακας Ελέγχου** — Dashboard with stats, building list, activity feed
- **Κτίρια** — Building cards with drill-down to units/expenses/bills
- **Ένοικοι** — Tenant directory with role filters and search
- **Κοινόχρηστα** — Billing overview with collection progress
- **Ανακοινώσεις** — Pinned announcements with priority badges
- **Συντήρηση** — Maintenance request tracker with status pipeline
- **Έγγραφα** — Document repository with category filters
- **Αναφορές** — Revenue charts, expense breakdowns, collection rates
- **Ρυθμίσεις** — Org details, billing config, notifications, team management
